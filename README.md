# Titre du TP
Application Android simple avec compteur et message Toast

---

# Objectif

L’objectif de ce TP est de créer une application Android simple utilisant :
- `TextView`
- `Button`
- `LinearLayout`
- Les événements de clic dans `MainActivity`

L’application réalise deux fonctionnalités :
1. Incrémenter un compteur affiché à l’écran.
2. Afficher un message Toast contenant le texte **"Bonjour"**.

---

# Structure du projet

```text
app/
├── java/
│   └── MainActivity.java
├── res/
│   └── layout/
│       └── activity_main.xml
```
